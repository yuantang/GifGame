# Now冥想10周年庆 - 大转盘抽奖

## 项目简介

这是一个基于 HTML/CSS/JS 实现的 1:1 设计稿还原的大转盘抽奖页面。

## 功能特点

- **1:1 还原**：完全使用设计稿切图，达到像素级视觉一致。
- **移动端适配**：支持全机型移动端浏览，自适应宽屏和竖屏。
- **精确校准**：中奖指向角度经过精准算法偏移计算，确保视觉与逻辑 100% 匹配。
- **自定义概率**：
  - 二等奖: 1.5%
  - 三等奖: 19%
  - 四等奖: 25%
  - 谢谢参与: 54.5%

## 部署说明

本项目已适配静态托管服务，推荐部署至 Gitee Pages。

## 运行方式

直接在浏览器打开 `index.html` 即可。
